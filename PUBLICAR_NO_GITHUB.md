# Como Publicar no GitHub

Este arquivo registra uma referência rápida para manter o portfólio localmente e publicar novas atualizações no GitHub.

## Atualizar este repositório

Depois de editar ou adicionar novos labs, execute:

```bash
git add .
git commit -m "Update cybersecurity lab portfolio"
git push
```

## Sugestão de Descrição do Repositório

Portfólio de laboratórios práticos em cibersegurança desenvolvidos durante minha pós-graduação, com estudos de caso, evidências, análise técnica e recomendações defensivas.

## Sugestão de Tópicos

```text
cybersecurity
wifi-security
kali-linux
wireshark
wpa2
ethical-hacking
security-lab
portfolio
```

## Próximos Labs

Uma boa estrutura para próximos estudos é:

```text
labs/
  nome-do-lab/
    README.md
    evidencias/
```

Cada `README.md` pode seguir a mesma organização: resumo, objetivos, ambiente, metodologia, resultados, limitações e recomendações de segurança.
