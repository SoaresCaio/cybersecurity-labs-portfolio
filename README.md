# Portfólio de Laboratórios em Cibersegurança

Este repositório reúne estudos de caso e laboratórios práticos desenvolvidos durante minha pós-graduação, com foco em segurança ofensiva, análise de redes, auditoria Wi-Fi e documentação técnica.

Todos os exercícios documentados aqui foram executados em ambientes controlados, com equipamentos próprios ou explicitamente autorizados. O objetivo é demonstrar raciocínio técnico, metodologia, análise de evidências e recomendações de mitigação.

## Sobre

Meu nome é Caio Adriano e este portfólio foi criado para registrar minha evolução prática em cibersegurança. Cada laboratório apresenta:

- Contexto e objetivo do experimento
- Ambiente utilizado
- Metodologia em alto nível
- Evidências e resultados observados
- Limitações encontradas
- Recomendações de segurança

## Competências Demonstradas

- Planejamento e execução de laboratório controlado
- Uso de Kali Linux em ambiente virtualizado
- Auditoria de redes Wi-Fi em 2.4 GHz
- Análise de tráfego com Wireshark
- Captura e interpretação de handshake WPA2
- Documentação técnica com evidências
- Comunicação de riscos e recomendações defensivas

## Ferramentas Utilizadas

- Kali Linux
- Oracle VirtualBox
- Adaptador TP-Link TL-WN722N v1.10
- Wireshark
- hostapd-mana
- dnsmasq
- aircrack-ng suite

## Aviso Ético

Os conteúdos deste repositório têm finalidade educacional e foram produzidos em ambiente de laboratório. As técnicas aqui descritas devem ser aplicadas somente em redes, sistemas e dispositivos próprios ou com autorização explícita.
